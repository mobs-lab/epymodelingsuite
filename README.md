# flumodelingsuite

Models for influenza forecasting and scenario modeling

## Installation

To install directly from GitHub (recommended for normal usage), run `pip install git+ssh://git@github.com/mobs-lab/flumodelingsuite.git`. You must have an SSH keypair for your machine enrolled with GitHub and have access to the [repo](https://github.com/mobs-lab/flumodelingsuite).

To install for development purposes, run `python -m pip install -e path/to/flumodelingsuite` for an editable install.
